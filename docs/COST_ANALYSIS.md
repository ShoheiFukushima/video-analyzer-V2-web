## 2. コスト計算式

### 2.1 Cloud Run コスト

```
Cloud Run コスト = (CPU単価 × vCPU数 × 処理秒数) + (メモリ単価 × GiB数 × 処理秒数)

例: 4vCPU, 4GiB, 120秒の場合
CPU: $0.00002400 × 4 × 120 = $0.01152
メモリ: $0.00000250 × 4 × 120 = $0.00120
合計: $0.01272
```

### 2.2 Whisper API コスト

```
Whisper コスト = (音声長さ[分]) × $0.006

例: 15秒の動画 (VADで10秒に削減)
10秒 = 0.167分
$0.006 × 0.167 = $0.001
```

### 2.3 Gemini Vision コスト

```
Gemini コスト = シーン数 × ($0.000084 + $0.00012)
             = シーン数 × $0.0002

例: 5シーンの場合
$0.0002 × 5 = $0.001
```

### 2.4 合計コスト

```
総コスト = Cloud Run + Whisper + Gemini + (R2/Turso/Vercel = $0)
```

## 5. 月間コスト試算

### 5.1 想定利用パターン

| プラン | 処理数/月 | 平均動画長 | 月間コスト |
|--------|----------|-----------|-----------|
| ライト | 100本 | 1分 | ~$1-2 |
| スタンダード | 500本 | 5分 | ~$10-20 |
| ヘビー | 2,000本 | 10分 | ~$80-150 |

### 5.2 無料枠の考慮

月間100本以下の処理であれば、ほぼ無料枠内で運用可能:
- Cloud Run: 100本 × 60秒 = 6,000秒 << 180,000秒 (無料枠)
- Gemini: 100本 × 10シーン = 1,000リクエスト << 1,500リクエスト/日
